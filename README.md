# Receita Federal Data Pipeline

## Overview
This project implements an end-to-end data engineering pipeline using open data
from the Brazilian Federal Revenue (CNPJ database).

This project idea comes from a LinkedIn user and i decided to test my skills with this to create a new portfolio.

## Data Source
Public monthly CSV datasets provided by the Receita Federal.

## Initial Architecture
Receita Federal  
   ↓  
Python download scripts  
   ↓  
Raw zone (CSV)  
   ↓  
PostgreSQL (staging)  
  
## Tech Stack
- Python
- PostgreSQL
- Docker
- GitHub
- WSL
